## Patchnotes 4.2.30

This update contains migrations to prepare and fix the data used by OpenSlides until version 4.2.29.
In addition to the migraitons, a Skript was added to request all data in JSON format.
All services remain unchanged except of the backend where the migrations have been added.

**This verion should _not_ be used to run OpenSlides. Only for data migration!**
