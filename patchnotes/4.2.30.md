## Patchnotes 4.2.30

This update contains migrations to prepare and fix the data used by OpenSlides until version 4.2.29.
In addition to the migraitons, a Skript was added to request all data in JSON format.
All services remain unchanged except of the backend where the migrations have been added.

**This verion should _not_ be be used to run OpenSlides. Only for data migration!**

### Migrations applied after running the update:
* (M74) Remove users without groups
* (M75) Remove broken projection content
* (M76) Fix Projector scroll postion
* (M77) Remove meeting users that have been deleted
* (M78) Remove non-existant or deleted meeting ids from motion/origin_meeting_id. Also removes duplicates from recommendation_extension_reference_ids.
* (M79) Remove empty strings from fields that'll be unique
* (M80) Edit out duplications that are left in the database from previous bugs
