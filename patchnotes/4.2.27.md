## Patchnotes 4.2.27

### New features
- Mandate check: New function integrated into the participant menu for mandate check. The function offers the option to statistically evaluate presence/absence in one or more groups. 

### Optimizations
- Improved accessibility for screen readers: Improvement of the structure on the autopilot and the presence display
- Translations added

### Bug fixes
- User profile: The entire button is highlighted.
- Elections > Sorting candidates: Manual sorting is saved correctly again. 
- Elections > Export: All elections are exported in full.
- Parliament mode > Speaking time projection: The correct speaking time is always displayed when changing the projection.
