## Patchnotes 4.2.25

### New Features
- Motions > Amendment recommendations: New function for extending and shortening existing change recommendations integrated.

### Optimizations
- Improved accessibility for screen readers: Improved structure on the autopilot, user profile, presence display, and polls.
- User profile: Presence indicator optimized. When presence is activated, it is displayed with a check mark to the left of the profile name.
- Deletion concept for participants: Users who were registered as submitters, editors, or spokespersons and have been deleted are listed in the fields as “Deleted users.”
- Authentication: The server time is used to check the expiration date of authentication tokens.
- UI optimization
- Translations added

### Bug Fixes
- Meetings > Duplicate: Meetings can be duplicated again.
- Motions > Export: Fixed error message when exporting motions with an empty motion list.
- Motions > Export-Dialog: Selected selection is retained again and is not reset.
- Elections > Voting with Yes/No/Abstain and 100% basis Yes/No: Display error in projection fixed. In the results projection, the values for abstentions are again correctly displayed as absolute values.
- Elections/Votes: Fixed an error that slowed down the opening of elections and motion votes.
