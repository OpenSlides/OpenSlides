## Patchnotes 4.3.0
### New Features
 - Deep Link: Integration of a deep link feature that allows users to access specific content (such as a motion) directly after logging in. (Example: https://demo.openslides.org/18/motions/1 - Logging in with the demo credentials opens the linked motion directly)
### Optimization
 - List of Speakers: Participant numbers and gender are now visible only to users with “Can manage agenda” permission.
 - Agenda > Import: Agenda items can now be imported with identical titles.
### Optimizes accessibility
- List of Speakers: All buttons have been provided with ARIA labels.
- Motions > List View: Navigation from a motion title (H1 heading) to the motion itself has been improved. When a title is selected, a motion can be opened by pressing Enter.
- Motions > Detail View: A new ARIA label has been added before the text of a motion or amendment, which indicates the position before the text via screen reader output.
- Elections: All buttons in the list of candidates have been provided with ARIA labels.
### Bug Fixes
- Dashboard > Calendar: Upcoming and past meetings are now sorted so that the events closest to the current date appear at the top.
- Dashboard > Settings: The “Save” button in the organization settings now works even if no time zone has been set.
- Committees > Detail View: Committee admins can now see all configured forwarding options, regardless of whether they have access to the source or the target committee. Existing access permissions are not affected by this.
- Committees > New meeting: In the meeting create form, the organization’s time zone is displayed correctly after reloading the page.
- Committees > New meeting / Edit a meeting: An error in the date picker has been fixed where the meeting date selected in the date picker was saved incorrectly. The date was always saved one day earlier than selected. Tip: Check the meeting date for meetings created or edited after the release of version 4.3.0 and correct the date if necessary.
- Committees > Duplicate Meeting: Meetings can now be duplicated again.
- Motions > Workflows: The button tooltip for a new workflow has been corrected.
- Participants > Multiple Selection > Delete: Deleting via multiple selection is now also possible even if your own account is included in the selection. When deleting, this account is skipped. Your own account cannot be deleted.
- iOS Devices > Projectors: Fixed a rendering error with projectors. All projections are now displayed correctly again.
